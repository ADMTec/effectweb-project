Descrição

Este é o guia de instalação da Effect Web 2.
Siga os passos corretamente para que não ocorra nenhum problema durante a instalação do sistema.
Dúvidas, problemas, entre demais coisas, entrar em contato com nossa equipe.

Requerimentos no servidor de hospedagem

Para o funcionamento correto do sistema é necessário que seu servidor de hospedagem tenha os seguintes requisitos:

  • » Versão do PHP: 5.2.x (não é com PHP 5.3 ou superiores)
  • » ionCube PHP Loader instalado (versão mínima: 4.0.4)
  • » Extensão MSSQL instalada
  • » Extensão cURL instalada
  • » Extensão Sockets instalada (opcional, porém recomendado)
  • » Extensão GD2 instalada
  • » Extensão ZIP instalada (opcional, porém recomendado)

Passo 1: Enviar arquivos

Envie todos os arquivos da Effect Web 2 que se encontra na pasta "Upload" do Package.

Passo 2: Liberar permissões

É necessário que libere permissões de leitura e escrita pelo FTP (em caso de Linux) ou pelo painel (em caso de Windows, tal como Plesk Panel).
Libere as permissões nos seguintes diretórios (incluíndo sub-diretórios):

  • » /cache/core_cache
  • » /cache/core_cache/license
  • » /cache/server_cache
  • » /cache/server_cache/db_ini
  • » /cache/server_cache/db_php
  • » /cache/server_cache/db_scripts
  • » /cache/server_cache/db_xml
  • » /cache/skin_cache
  • » /cache/skin_cache/templates
  • » /cache/temp_cache
  • » /control/Logs
  • » /control/Settings.php
  • » /public/style_css
  • » /public/style_images
  • » /public/style_resources
  • » /public/uploads

Passo 3: Configuração do controle

Efetue as configurações da pasta "control" do sistema nos seguintes passos:

  • » Envie o Item.txt (ou Item(kor).txt) localizado no seu MuServer para /control/Data/Items com o nome de Item.txt
  • » Configure a Settings.php do sistema localizado em /control/ (não é necessário inserir dados do SQL)
  • » Configure os arquivos localizados em /control/Data

Passo 4: Instalação do sistema

Os arquivos estão prontos para iniciar a instalação.
Acesse o sistema pelo AdminCP (www.exemplo.com/admin) para abrir o instalador e seguir os passos.
