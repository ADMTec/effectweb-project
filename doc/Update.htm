Descrição

Este é o guia de atualização da Effect Web 2.
Siga os passos corretamente para que não ocorra nenhum problema durante a atualização do sistema.
Dúvidas, problemas, entre demais coisas, entrar em contato com nossa equipe.

Requerimentos no servidor de hospedagem

Para o funcionamento correto do sistema é necessário que seu servidor de hospedagem tenha os seguintes requisitos:

  • » Versão do PHP: 5.2.x (não é com PHP 5.3 ou superiores)
  • » ionCube PHP Loader instalado (versão mínima: 4.0.4)
  • » Extensão MSSQL instalada
  • » Extensão cURL instalada
  • » Extensão Sockets instalada (opcional, porém recomendado)
  • » Extensão GD2 instalada
  • » Extensão ZIP instalada (opcional, porém recomendado)

Passo 1: Enviar arquivos

Envie todos os arquivos da Effect Web 2 que se encontra na pasta "Upload" do Package com exceção dos seguintes:

  • cache/
  • control/Data/
  • control/Settings.php

Passo 2: Permissão ao atualizador

Este passo poderá ser ignorado caso já tenha sido feito ao instalar/atualizar o sistema antes.
Adicione na Settings.php, na linha $CTM_SETTINGS['ADMINCONTROLPANEL']['SYSTEM_MANAGE'] o login da sua conta para ter acesso ao atualizador.

Passo 3: Atualização do sistema

Os arquivos estão prontos para iniciar a atualização.
Acesse o sistema pelo AdminCP (www.exemplo.com/admin) para abrir o atualizador e seguir os passos.
É de extrema importância que os passos devem ser seguidos com máxima de atenção possível para que não ocorra erros durante a atualização.
